# Project Inventory App

A full-stack **Inventory Management Web Application** built as part of **The Odin Project** curriculum.

This app allows users to create and manage **categories** and **projects**, demonstrating full CRUD functionality using **Node.js, Express, PostgreSQL, and EJS**.

---

## üöÄ Features

### Categories
- Create new categories
- View all categories
- View a single category
- Update category details
- Delete categories

### Projects
- Create projects under a category
- View all projects
- View a single project
- Update project details
- Delete projects
- Assign projects to categories using a dropdown selector

### UI / UX
- Server-side rendering with **EJS**
- Reusable layout and partials (navbar)
- Clean, modern styling with a global CSS file
- Pre-filled forms for editing records
- Graceful empty states
